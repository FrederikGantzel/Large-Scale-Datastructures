No special modules are needed to run the program.

The program requires inputs of the following format:

	./homework Problem<subproblem> <number of random sequences from the COVID genome>

For example, to run the programs answers for subproblem 1B, and make 50,000 random sequences, the inputs would be:

	./homework Problem1B 50000

The names for each of the subproblems are:
Problem1A
Problem1B
